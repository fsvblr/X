Welcome to JooYii Project.

This is Joomla!CMS Library that is built by @joomplace developers to bring new life to MVC structure of Joomla!CMS.

Inspired by greatest

This is not something that would provide tons of new code and some not yet known tech. This is about taking a good approach from anywhere and adapting it to Joomla!CMS core.

Actually at version 1.0 it's mostly Doctrine, Yii2, Joomla!CMS PSR-4 convention.

Built to improve

Everybody are welcome to use and to PR this library.

Just one thing to keep in mind when PR:

If you PR this repo - it mostly means you use the code. And it mostly means that you like the code. So it must mean that you trust maintainers. So, no yelling, if some PR will not make it.
